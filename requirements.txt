# HOW TO USE:
#   pip install -r requirements.txt
#   python tests/smoke_test.py   <- run this after installing to verify
#
# PINNING STRATEGY:
#   Exact versions are pinned to prevent "works on my machine" problems.

#
# KNOWN BREAKAGE POINTS TO WATCH:
#   1. pydantic v1 vs v2: FastAPI 0.100+ requires pydantic v2. Do NOT

#   2. azure-cognitiveservices-speech: installs an OS-specific wheel.
#      Windows gets the Windows wheel automatically. If install fails,
#      check https://pypi.org/project/azure-cognitiveservices-speech/
#   3. openai: use the modern AzureOpenAI() client (1.x). The old
#      openai.ChatCompletion.create() style is removed in 1.x.
# ==============================================================================
